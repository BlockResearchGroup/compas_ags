Scene

class compas_ags.rhino.Scene[source]

Bases: object

A Rhino scene for AGS.

Attributes

objects (dict) – Mapping between GUIDs and diagram objects added to the scene. The GUIDs are automatically generated and assigned.

Examples

from compas_ags.diagrams import FormGraph
from compas_ags.diagrams import FormDiagram

from compas_ags.rhino import Scene

graph = FormGraph.from_obj(FILE)
form = FormDiagram.from_graph(graph)

scene = Scene()

guid = scene.add(form, name="Form", layer="AGS::FormDiagram")
form = scene.find(guid)

scene.clear()
scene.update()
