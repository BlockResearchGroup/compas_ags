Contributions

We accept code contributions through pull requests. In short, this is how that works.

  1. Fork a copy of the the compas_ags repository to your account and clone the fork to your computer.

  2. Create a virtual environment using your tool of choice (e.g. virtualenv, conda, etc).

  3. Install development dependencies:

$ pip install -r requirements-dev.txt

  1. Make a branch for your contribution using a descriptive name.

  2. Make the required changes to the code of the branch.

  3. Add yourself to CONTRIBUTORS.md.

  4. Commit your changes and push your branch to GitHub.

  5. Create a pull request through the GitHub website.

During development, use pyinvoke tasks on the command line to ease recurring operations:

  • invoke clean: Clean all generated artifacts.

  • invoke check: Run various code and documentation style checks.

  • invoke docs: Generate documentation.

  • invoke: Show available tasks.
