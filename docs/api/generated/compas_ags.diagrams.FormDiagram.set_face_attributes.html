FormDiagram.set_face_attributes

FormDiagram.set_face_attributes(key, names, values)

Set multiple attributes of one face.

Parameters

  • key (hashable) – The identifier of the face.

  • names (list of str) – The names of the attributes.

  • values (list of object) – The new values of the attributes.
