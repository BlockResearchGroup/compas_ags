FormDiagram.gkey_key

FormDiagram.gkey_key(precision='3f')

Returns a dictionary that maps geometric keys of a certain precision to the keys of the corresponding vertices.

Parameters

precision (str (3f)) – The float precision specifier used in string formatting.

Returns

dict – A dictionary of geometric key-key pairs.
